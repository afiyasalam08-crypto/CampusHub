# 🎓 CampusHub

## 📖 Overview

CampusHub is a web-based college campus management platform developed using React.js, Tailwind CSS, and Firebase Authentication. The platform provides students with a centralized system for accessing academic resources, campus updates, events, student communities, and personal academic tools.

The primary objective of CampusHub is to improve campus communication, student engagement, and resource accessibility through a modern, responsive, and user-friendly web application.

---

## ✨ Features

### 🔐 Authentication

* Secure user registration and login
* Firebase Authentication integration
* Session management and logout functionality

### 🏠 Dashboard

* Centralized student dashboard
* Quick access to announcements, events, notes, clubs, and academic tools
* Responsive and intuitive user interface

### 📢 Announcements

* Display important college notices and updates
* Academic and administrative announcements
* Centralized information board for students

### 📅 Events Management

* View upcoming workshops, seminars, and cultural events
* Event registration interface
* Organized event listings with detailed information

### 📚 Notes Sharing

* Access academic study materials
* Subject-wise organization of resources
* Downloadable learning materials
* Scalable design for future file upload support

### 👥 Clubs and Communities

* Browse student clubs and organizations
* View club information and activities
* Encourage student participation and engagement

### 💬 Student Forum

* Academic discussions and peer interaction
* Question-and-answer support
* Sharing opportunities and campus-related information

### 👤 Student Profile

* Manage personal and academic information
* Update department, academic year, and skills
* Maintain professional links such as LinkedIn and GitHub
* Editable profile management system

### 📊 CGPA Calculator

* Dynamic subject and credit management
* Grade-point based CGPA calculation
* Circular progress visualization
* Academic performance analysis and feedback

## 🎯 Project Objective

CampusHub aims to create a unified digital platform that enhances the student experience by providing seamless access to academic resources, campus activities, and collaborative communities.

The system is designed to streamline communication, improve accessibility, and support student engagement within a modern campus environment.
